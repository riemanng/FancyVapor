# FancyVapor

Back-End for [Smart Home iOS app](https://github.com/riemanng/SmartHome)

## Tech Stack

- Swift
- Vapor
- Fluent ORM
- PostgreSQL
- Locust (for load testing)
- nginx (on Ubuntu)

## Technical requirenments

Ubuntu 20.4 LTS or macOS BigSur 11.2.3
